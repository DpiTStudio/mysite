# DPIT-CMS: Система управления контентом на Django

Добро пожаловать в репозиторий проекта **DPIT-CMS**. Это современная, функциональная CMS, разработанная для управления корпоративным сайтом, портфолио и услугами.

## 🚀 Основные возможности

- **Управление контентом**: Динамические страницы, новости с автогенерацией, гибкое портфолио.
- **Инструменты продаж**: Каталог услуг с тарификацией, корзина заказов, система отзывов с модерацией.
- **Поддержка пользователей**: Встроенная система тикетов и личные кабинеты.
- **Администрирование**: Премиальная панель управления на базе `Jazzmin`, мониторинг логов, система бэкапов.
- **Автоматизация**: Автоматическое переименование медиа-файлов, отправка уведомлений, интеграция с Redis/Celery.

## 📂 Структура проекта

Ниже приведено описание основных разделов проекта. Каждая папка содержит свой детальный `README.md`.

### Ядро и настройки
- **[mysite/](./mysite/)** — Системные настройки проекта, конфигурация URL, задачи бэкапа и Celery.
- **[main/](./main/)** — Ядро системы: глобальные настройки сайта, статические страницы, утилиты.

### Пользователи и общение
- **[accounts/](./accounts/)** — Расширенная модель пользователя, регистрация, вход (в т.ч. через соцсети).
- **[tickets/](./tickets/)** — Техническая поддержка и взаимодействие с клиентами.
- **[mail/](./mail/)** — Утилиты для работы с почтой (проверка входящих по IMAP, тест SMTP).

### Контент и услуги
- **[news/](./news/)** — Модуль новостей с системой автоматических уведомлений о событиях на сайте.
- **[portfolio/](./portfolio/)** — Демонстрация выполненных работ.
- **[services/](./services/)** — Каталог услуг с гибким управлением ценами и поиском.
- **[knowledge_base/](./knowledge_base/)** — База знаний (Вики) с руководствами и статьями.

### Интерактив
- **[cart/](./cart/)** — Корзина заказов для услуг.
- **[favorites/](./favorites/)** — Система "Избранного" для любого типа контента.
- **[reviews/](./reviews/)** — Сбор и модерация отзывов клиентов.

### Системные ресурсы
- **[static/](./static/)** — Стили (CSS), скрипты (JS) и статические изображения.
- **[templates/](./templates/)** — HTML-шаблоны сайта (Django Template Language).
- **[media/](./media/)** — Пользовательский контент (фото работ, аватары).
- **[logfiles/](./logfiles/)** — Управление серверными логами через админку.

## 🛠 Технологический стек

- **Backend**: Django 4.x, Python 3.10+
- **Database**: SQLite (разработка) / PostgreSQL (продакшн)
- **Frontend**: Bootstrap 5, HTMX, TinyMCE
- **Async & Tasks**: Redis, Celery (резервное копирование)
- **Security**: django-simple-captcha, django-allauth

## 🔧 Быстрый старт

1. **Окружение**:
   Создайте файл `.env` на основе примера и заполните секретные ключи и настройки почты.
2. **Зависимости**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Миграции**:
   ```bash
   python manage.py migrate
   ```
4. **Запуск**:
   ```bash
   python manage.py runserver
   ```

Подробную информацию по каждому модулю ищите в соответствующих папках.
