# Django проект "mysite"

Этот каталог содержит исходный код Django-приложения.

## Основные компоненты

- `manage.py` — Инструмент командной строки для взаимодействия с проектом.
- `mysite/` — Настройки проекта (settings.py, urls.py, wsgi.py).
- `db.sqlite3` — База данных проекта.
- `requirements.txt` — Список зависимостей Python.

## Приложения

### [Main](./main/)
Центральное приложение, содержащее:
- `SiteSettings` — Глобальные настройки (контакты, логотипы).
- `Page` — Управление статическими страницами.
- `utils.py` — Утилиты для переименования файлов при загрузке.
- `context_processors.py` — Процессоры контекста для доступности настроек во всех шаблонах.

### [Accounts](./accounts/)
Управление пользователями:
- Расширенная модель `User` с полями для телефона и аватара.

### [News](./news/)
Информационный модуль:
- Категории новостей.
- Публикации с поддержкой HTML и изображений.

### [Portfolio](./portfolio/)
Демонстрация работ:
- Проекты и кейсы, разбитые по категориям.

### [Reviews](./reviews/)
Обратная связь:
- Отзывы клиентов с системой премодерации.

### [Tickets](./tickets/)
Техническая поддержка:
- Создание тикетов пользователями и ответы администраторов.

### [Mail](./mail/)
Почтовый клиент:
- Чтение входящих писем (IMAP).
- Проверка отправки (SMTP).

### [Logfiles](./logfiles/)
Управление журналами:
- Мониторинг логов и создание резервных копий через админку.

## Статика и Шаблоны

- `static/` — Глобальные стили и скрипты.
- `templates/` — Общие шаблоны (base.html, header.html, footer.html).
- `media/` — Загруженные пользователями файлы (изображения, документы).

## Логирование

Система записывает логи в папку `logfiles/`. Основные файлы:
- `debug.log` — Отладочная информация.
- `error.log` — Ошибки системы.
