# Приложение Accounts

Отвечает за аутентификацию, регистрацию и управление профилями пользователей.

## Файлы приложения

### `models.py`
- `User` — кастомная модель пользователя, унаследованная от `AbstractUser`. Добавляет поля:
    - `phone` (Телефон)
    - `avatar` (Аватар)
    - `created_at` / `updated_at` (метки времени через `TimestampModel`).

### `views.py`
- `register_view(request)` — регистрация новых пользователей.
- `login_view(request)` — авторизация. Поддерживает перенаправление (next parameter).
- `logout_view(request)` — выход из системы.
- `profile_view(request)` — личный кабинет пользователя, где можно редактировать свои данные, менять аватар и телефон.

### `forms.py`
- `UserRegistrationForm` — форма регистрации с валидацией пароля.
- `UserLoginForm` — форма входа на базе стандартной `AuthenticationForm`.
- `UserProfileForm` — форма для редактирования данных в личном кабинете.

### `urls.py`
Маршруты для входа, выхода, регистрации и профиля. Использует пространство имен `accounts`.
