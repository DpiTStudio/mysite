# Резервные копии (Backups)

В этом каталоге хранятся автоматически создаваемые резервные копии проекта.

## Типы бэкапов

1. **База данных (`db/`)**: Дампы базы данных SQLite (`db.sqlite3`). Создаются Celery-задачами по расписанию.
2. **Медиа-файлы (`media/`)**: ZIP-архивы содержимого папки `/media/`.
3. **Полные бэкапы**: Объединенные архивы кода, базы и медиа.

---

## Настройки (в `settings.py`)
- `BACKUP_MAX_DB_COUNT`: Лимит количества хранимых копий базы данных (старые удаляются автоматически).
- `BACKUP_MAX_MEDIA_COUNT`: Лимит количества хранимых архивов медиа.

Задачи по управлению бэкапами описаны в файле `mysite/backup_tasks.py`.
