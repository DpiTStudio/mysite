# Документация приложения Cart

Приложение `cart` реализует функционал корзины покупок для услуг и товаров сайта.

## Обзор
Корзина построена на базе Django сессий, что позволяет пользователям добавлять услуги в корзину без необходимости моментальной авторизации. Корзина поддерживает добавление, удаление, изменение количества и просмотр списка выбранных услуг.

---

## Описание файлов

### 1. `cart.py`
Основной класс `Cart`, управляющий состоянием в сессии.
- **Логика:** Хранит данные в словаре внутри `request.session`.
- **Методы:**
    - `add()`: Добавление товара или обновление количества.
    - `save()`: Пометка сессии как измененной.
    - `remove()`: Удаление товара из корзины.
    - `clear_session()`: Полная очистка корзины.
    - `get_total_price()`: Подсчет суммарной стоимости всех позиций.
- **Итерация:** Позволяет обходить объект корзины в цикле, автоматически подгружая связанные объекты `Service` из БД.

### 2. `context_processors.py`
- `cart(request)`: Контекстный процессор, делающий объект корзины доступным во всех шаблонах сайта (через переменную `cart`). Позволяет отображать количество товаров в шапке сайта на любой странице.

### 3. `views.py`
- `cart_add`: Обработчик добавления (POST).
- `cart_remove`: Обработчик удаления.
- `cart_detail`: Страница просмотра содержимого корзины и оформления заказа.

### 4. `forms.py`
- `CartAddProductForm`: Форма для добавления товара с выбором количества (используется на страницах услуг).

### 5. `urls.py`
Маршруты.
- `/` -> Детали корзины.
- `/add/<int:service_id>/` -> Добавление.
- `/remove/<int:service_id>/` -> Удаление.

### 6. `apps.py`
- `verbose_name = "Корзина"`: Имя приложения в админ-панели.
