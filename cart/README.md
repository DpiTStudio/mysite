# Документация приложения Cart

Приложение `cart` реализует функционал корзины покупок для услуг сайта.

## Обзор
Корзина построена на базе Django сессий, что позволяет пользователям добавлять услуги в корзину без необходимости моментальной авторизации. Корзина поддерживает добавление, удаление и просмотр списка выбранных услуг.

---

## Описание файлов

### 1. `apps.py`
Конфигурация приложения.
- `verbose_name = "Корзина"`: Имя в админке.

### 2. `cart.py`
Основной класс `Cart`.
- Реализует логику добавления, удаления, сохранения и очистки корзины в сессии.
- Позволяет итерироваться по элементам корзины, автоматически подгружая объекты `Service` из базы данных.
- Подсчитывает общее количество элементов и суммарную стоимость.

### 3. `context_processors.py`
- `cart(request)`: Контекстный процессор, делающий объект корзины доступным во всех шаблонах сайта (например, для отображения количества товаров в шапке).

### 4. `views.py`
- `cart_add`: Добавление услуги в корзину.
- `cart_remove`: Удаление услуги из корзины.
- `cart_detail`: Страница просмотра содержимого корзины.

### 5. `urls.py`
Маршруты URL.
- `/` -> Просмотр корзины.
- `/add/<service_id>/` -> Добавление.
- `/remove/<service_id>/` -> Удаление.
