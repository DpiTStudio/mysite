# Документация проекта (docs)

В этом каталоге собраны полезные инструкции и отчеты о проделанной работе.

## Файлы

- `SETUP_INSTRUCTIONS.md` — Подробное руководство по установке и настройке проекта с нуля.
- `SETUP_COMPLETE.md` — Краткий отчет о завершении первичной настройки.
- `FIXES_APPLIED.md` — Список исправленных ошибок и багов в системе.
- `IMPROVEMENTS.md` — Информация о внедренных улучшениях и новом функционале.
- `VSCODE_SETUP.md` — Настройка рабочего окружения в Visual Studio Code.
- `mail.md` — Информация о доступах к почтовому серверу (Timeweb).
