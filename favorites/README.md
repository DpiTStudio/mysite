# Документация приложения Favorites

Приложение `favorites` реализует систему "Избранного" (или закладок), позволяя пользователям сохранять любые объекты сайта (новости, работы из портфолио, услуги) для быстрого доступа.

## Обзор
Система построена с использованием `Generic Foreign Key` (Django ContentTypes), что делает её универсальной и позволяет добавлять в избранное объекты любых моделей без изменения их структуры.

---

## Описание файлов

### 1. `models.py`
Содержит основную логику хранения данных.

**Классы:**
- `Favorite`: Модель связи пользователя с объектом.
    - **Поля:**
        - `user`: Ссылка на пользователя (`accounts.User`).
        - `content_type`: Ссылка на тип модели объекта.
        - `object_id`: ID конкретного объекта.
        - `content_object`: Прямая ссылка на объект (через GenericForeignKey).
        - `created_at`: Дата добавления в избранное.
    - **Особенности:** Используется уникальность по полям `user`, `content_type`, `object_id`, чтобы один пользователь не мог добавить один и тот же объект в избранное дважды.

### 2. `views.py`
Логика взаимодействия пользователя с избранным.

**Функции:**
- `toggle_favorite(request, app_label, model_name, object_id)`:
    - Универсальный обработчик (POST). Принимает параметры модели и ID.
    - Если объект уже в избранном — удаляет его. Если нет — создает запись.
    - Возвращает JSON ответ с текущим состоянием (is_favorite) и общим количеством "лайков" у объекта.
- `favorites_list(request)`:
    - Отображает страницу со всеми избранными объектами текущего пользователя.
    - Группирует объекты по их типам (например отдельно новости, отдельно услуги) для удобства отображения.

### 3. `urls.py`
Маршрутизация.
- `toggle/<app_label>/<model_name>/<int:object_id>/` -> Переключение состояния.
- `/` -> Список избранного.

### 4. `templatetags/` (если есть)
Обычно содержит теги для проверки, находится ли объект в избранном, прямо внутри HTML-шаблона.

## Примеры использования (HTMX / AJAX)
Для добавления кнопки "В избранное" используется POST запрос к `toggle_favorite`. В проекте часто интегрируется с HTMX для обновления иконок без перезагрузки страницы.
