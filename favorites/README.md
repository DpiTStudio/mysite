# Документация приложения Favorites (Избранное)

Приложение `favorites` реализует универсальную систему "лайков" или добавления в избранное для любого типа контента на сайте (новости, работы, услуги).

## Обзор
Система построена на базе **Generic Foreign Keys** (Django ContentTypes), что позволяет привязывать реляционную связь к объекту любой модели без изменения структуры самой модели. Пользователи могут сохранять интересующий их контент в личном списке "Избранное" для быстрого доступа в будущем.

---

## Описание файлов

### 1. `admin.py`
Настройка административной панели.
- `FavoriteAdmin`: Просмотр списка всех закладок пользователей.
- Позволяет фильтровать по пользователю, дате добавления и типу объекта.

### 2. `apps.py`
Конфигурация приложения.
- `verbose_name = "Избранное"`: Человекочитаемое имя приложения.

### 3. `models.py`
Структура данных.

**Классы:**

#### `Favorite`
Центральная модель системы.
- **Поля:**
    - `user`: Ссылка на пользователя (владельца закладки).
    - `content_type`: Тип модели (к чему ставится лайк).
    - `object_id`: Индитификатор конкретного объекта.
    - `content_object`: Прямая ссылка на объект через Generic FK.
    - `created_at`: Время добавления.
- **Особенности:** Индексированное ограничение `unique_together` гарантирует, что пользователь не может добавить один и тот же объект в избранное дважды.

### 4. `views.py`
Логика преставлений.
- `toggle_favorite`: AJAX-обработчик (POST), который добавляет объект в избранное, если его там нет, или удаляет, если он уже есть (тоггл-логика).
    - Возвращает JSON: `{ "is_favorite": bool, "count": int }`.
    - Используется в шаблонах через HTMX или обычный JS.
- `favorites_list`: Страница со списком всех избранных объектов текущего пользователя, сгруппированных по типу (Новости, Работы, Услуги).

### 5. `urls.py`
Маршруты URL.
- `/` -> Список избранного.
- `toggle/<str:app_label>/<str:model_name>/<int:object_id>/` -> API для переключения статуса избранного.

---

## Использование в шаблонах
Для добавления кнопки "Избранное" к любому объекту (например, новости) используется HTMX запрос к пути `toggle`:

```html
<button hx-post="{% url 'favorites:toggle' 'news' 'news' news.id %}" ...>
    ...
</button>
```
