# Документация приложения Knowledge Base

Приложение `knowledge_base` реализует систему "Базы знаний" или "Вики" проекта, позволяя создавать структурированные статьи и руководства.

## Обзор
Приложение позволяет организовывать контент по категориям и подкатегориям, поддерживает поиск по статьям и динамическое отображение популярных материалов.

---

## Описание файлов

### 1. `admin.py`
Настройка административной панели.
- `CategoryAdmin`: Управление категориями базы знаний. Включает поля для иконок и SEO.
- `ArticleAdmin`: Управление статьями. Интегрировано с TinyMCE для форматирования текста. Поддерживает управление статусами публикации.

### 2. `apps.py`
Конфигурация приложения.
- `verbose_name = "База знаний"`: Имя в админке.

### 3. `models.py`
Структура данных.

**Классы:**
- `Category`: Категории статей. Имеют заголовок, описание, иконку и слаг.
- `Article`: Основная модель статьи.
    - **Поля:** Категория, заголовок, контент (HTML), автор, просмотры, теги.
    - **Методы:** Учет просмотров, формирование URL.

### 4. `views.py`
Логика представлений.
- `ArticleListView`: Список всех статей с возможностью фильтрации по категориям.
- `ArticleDetailView`: Просмотр контента статьи.
- `CategoryDetailView`: Список статей в конкретной категории.
- `SearchView`: Полнотекстовый поиск по статьям.

### 5. `urls.py`
Маршруты URL.
- `/` -> Главная страница базы знаний.
- `/category/<slug>/` -> Просмотр категории.
- `/<slug>/` -> Чтение статьи.
- `/search/` -> Поиск.
