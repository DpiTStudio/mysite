# Приложение Mail

Модуль для управления электронной почтой через веб-интерфейс сайта. Позволяет просматривать входящие сообщения и отправлять тестовые письма для проверки настроек SMTP.

## Файлы приложения

### `views.py`
Основные функции управления почтой:
- `is_staff(user)` — функция проверки прав (только для персонала).
- `mail_index(request)` — главная страница почтового модуля. 
    - Подключается к IMAP-серверу (настройки из `settings.py`).
    - Извлекает последние 10 сообщений из папки INBOX.
    - Декодирует заголовки (тема, отправитель) и отображает их в списке.
- `send_test_email(request)` — отправка тестового сообщения через SMTP для проверки работоспособности почтового сервера.

### `urls.py`
Маршруты для доступа к индексу почты и отправке тестовых писем. Использует пространство имен `mail`.

### `templates/mail/`
- `index.html` — интерфейс для просмотра списка писем и формы отправки тестового email.

## Настройки (Settings)
Для работы приложения используются следующие параметры в `settings.py`:
- `EMAIL_HOST`, `EMAIL_PORT`, `EMAIL_HOST_USER`, `EMAIL_HOST_PASSWORD` — для отправки (SMTP).
- `IMAP_HOST`, `IMAP_PORT`, `IMAP_USER`, `IMAP_PASSWORD` — для чтения (IMAP).
