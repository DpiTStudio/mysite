# Медиа-файлы (Media)

Каталог для хранения файлов, загруженных пользователями и администраторами через интерфейс сайта.

## Структура

- `avatars/`: Фотографии профилей пользователей.
- `news/`: Изображения к новостным статьям.
- `portfolio/`: Фотографии выполненных работ и иконки услуг.
- `pages/`: Изображения для статических страниц.
- `headers/`: Фоновые изображения для "шапок" разделов сайта.

---

## Особенности обработки
Все файлы, загружаемые в этот каталог, проходят автоматическую обработку утилитой `RenameUploadTo`:
1. **Транслитерация**: Русские имена файлов меняются на английские.
2. **Уникальность**: К имени добавляется дата и время загрузки для предотвращения перезаписи существующих файлов.
3. **Организация**: Файлы автоматически раскладываются по подпапкам в зависимости от их типа.

**Важно**: Этот каталог обычно исключен из Git (`.gitignore`). Регулярно делайте резервные копии этой папки, так как в ней хранятся ценные данные пользователей.
