# Конфигурация проекта (mysite/mysite)

Этот каталог содержит основные настройки Django-проекта.

## Файлы

### `settings.py`
Глобальные настройки проекта:
- Конфигурация базы данных (по умолчанию SQLite).
- Настройки установленных приложений (INSTALLED_APPS).
- Настройки статических и медиа файлов.
- Параметры логирования (LOGGING).
- Настройки TinyMCE и Jazzmin (тема админки).
- Определение кастомной модели пользователя (`AUTH_USER_MODEL`).

### `urls.py`
Главный файл маршрутизации проекта. Здесь подключаются URL-адреса всех приложений:
- `admin/` — панель администратора.
- `tinymce/` — редактор контента.
- `accounts/` — аутентификация.
- `news/` — новости.
- `portfolio/` — портфолио.
- `reviews/` — отзывы.
- `tickets/` — техподдержка.
- `/` — главная страница и статические страницы (приложение `main`).

### `wsgi.py` и `asgi.py`
Файлы для развертывания проекта на веб-серверах (Apache, Nginx, Gunicorn и т.д.).
- `wsgi.py` — для синхронных серверов.
- `asgi.py` — для асинхронных серверов и поддержки протоколов типа WebSockets.
