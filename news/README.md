# Приложение News

Модуль для публикации новостей, статей и объявлений на сайте.

## Файлы приложения

### `models.py`
- `NewsCategory` — категории новостей (например, "Мероприятия", "Технологии"). Поддерживает иконки, описание и SEO-настройки.
- `News` — сама публикация. Содержит заголовок, изображение, HTML-контент, счетчик просмотров и SEO-поля.
- `Comment` — простая модель комментариев к новостям.

### `views.py`
- `get_latest_news()` — вспомогательная функция с кешированием для получения последних новостей.
- `news_list(request)` — общий список новостей с пагинацией (9 элементов на страницу).
- `news_by_category(request, category_slug)` — фильтрация новостей по конкретной категории.
- `news_search(request)` — полнотекстовый поиск по заголовкам, описанию и контенту новостей.
- `news_detail(request, slug)` — детальная страница новости. При каждом просмотре вызывается `increment_views()`.

### `urls.py`
Маршруты для списка, категорий, поиска и детального просмотра новостей.

### `signals.py`
Используется для автоматизации (например, очистка кеша при обновлении новостей).

### `templates/news/`
- `list.html` — шаблон списка новостей (используется также для поиска и категорий).
- `detail.html` — шаблон детальной страницы новости.
