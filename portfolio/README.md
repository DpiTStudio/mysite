# Приложение Portfolio

Предназначено для демонстрации выполненных работ, кейсов и проектов.

## Файлы приложения

### `models.py`
- `PortfolioCategory` — категории работ (например, "Веб-дизайн", "Маркетинг"). Содержит SEO-поля и настройки шапки.
- `Portfolio` — конкретный кейс или проект. Включает заголовок, изображение, детальное описание (HTML) и счетчик просмотров.

### `views.py`
- `get_latest_portfolio()` — возвращает 3 последних добавленных проекта.
- `portfolio_list(request)` — отображает все активные проекты с пагинацией.
- `portfolio_by_category(request, category_slug)` — фильтрация проектов по категории.
- `portfolio_detail(request, slug)` — детальная страница кейса. Реализован инкремент просмотров и вывод похожих проектов из той же категории.

### `urls.py`
Маршруты для доступа к списку работ и детальным страницам.

### `signals.py`
Содержит логику для автоматизации, например, создание новости при добавлении нового проекта в портфолио.

### `templates/portfolio/`
- `list.html` — список проектов с фильтрацией по категориям.
- `detail.html` — страница с полным описанием кейса.
