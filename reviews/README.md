# Приложение Reviews

Система сбора и отображения отзывов клиентов.

## Файлы приложения

### `models.py`
- `Review` — модель отзыва. Поля:
    - `full_name` (ФИО автора)
    - `phone`, `email` (контакты)
    - `content` (текст отзыва)
    - `status` (статус модерации: На модерации, Одобрено, Отклонено). По умолчанию — `pending`.

### `views.py`
- `reviews_list(request)` — отображает список только одобренных администратором отзывов.
- `create_review(request)` — страница с формой для добавления нового отзыва. После успешной отправки выводится сообщение о необходимости модерации.

### `forms.py`
- `ReviewForm` — форма для ввода данных отзыва с фронтенда.

### `admin.py`
Настроено управление отзывами: администратор может менять статус прямо из списка (list_editable).

### `signals.py`
Автоматизация: при одобрении отзыва может создаваться соответствующая новость на сайте.
