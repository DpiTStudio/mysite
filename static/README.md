# Статические файлы (Static)

Каталог для хранения файлов, которые не меняются в процессе работы приложения (CSS, JS, иконки).

## Структура

- `css/`: Стили сайта.
    - `style.css`: Основной файл стилей проекта.
    - `admin_custom.css`: Кастомные стили для административной панели Jazzmin.
- `js/`: Скрипты JavaScript.
    - Код для инициализации анимаций, обработки форм и взаимодействия с HTMX.
- `images/`: Статические изображения интерфейса.
    - Логотипы, иконки, фавиконы.
- `vendor/`: Библиотеки сторонних разработчиков (Bootstrap, FontAwesome и др.), если они подключены локально.

---

## Сборка и обслуживание
В режиме разработки Django отдает файлы напрямую из этой папки. Перед выкладкой на сервер (в продакшн) необходимо выполнить команду:
```bash
python manage.py collectstatic
```
Это соберет все файлы в папку `staticfiles` для эффективной отдачи веб-сервером (Nginx/WhiteNoise).
