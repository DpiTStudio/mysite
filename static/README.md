# Статические файлы (Static)

Этот каталог содержит все глобальные ресурсы проекта, необходимые для работы фронтенд-части сайта: стили, скрипты, шрифты и системные иконки.

---

## 📂 Структура каталога

### 1. `css/` — Каскадные таблицы стилей
- `main.css`: Главный файл стилей проекта с переменными дизайна (цвета, отступы).
- `bootstrap.min.css`: Скомпилированная версия фреймворка Bootstrap 5.
- `responsive.css`: Настройки адаптивности под мобильные устройства и планшеты.
- `themes/`: Подкаталог для сезонных тем оформления (зима, лето и др.).

### 2. `js/` — Скрипты JavaScript
- `main.js`: Основная логика интерфейса (инициализация плагинов, анимации).
- `htmx.min.js`: Библиотека для работы с динамическими запросами без перезагрузки страниц.
- `cart.js`: Логика взаимодействия с корзиной и применения промокодов.
- `scripts/`: Сторонние плагины (TinyMCE, Flatpickr и др.).

### 3. `images/` — Графические ресурсы
- `logos/`: Варианты логотипа компании (светлый, темный, SVG).
- `icons/`: Системные иконки интерфейса, не связанные с пользовательским контентом.
- `placeholders/`: Заглушки для объектов без загруженных изображений.
- `backgrounds/`: Статические фоновые изображения для разделов сайта.

---

## ⚙️ Системные файлы в корне static

- `manifest.json`: Описание веб-приложения для поддержки PWA (Mobile Install).
- `sw.js`: Service Worker для кэширования ресурсов и ускорения повторной загрузки страниц.
- `robots.txt`: Статическая версия файла инструкций для поисковых роботов (обычно отдается динамически из `main`).
- `favicon.ico`: Значок сайта в адресной строке браузера.

---

## 💡 Рекомендации по разработке
1. **Сборка**: Для оптимизации рекомендуется использовать минифицированные версии библиотек (`.min.js`, `.min.css`).
2. **Collectstatic**: После внесения изменений на боевом сервере обязательно выполните команду:
   ```bash
   python manage.py collectstatic
   ```
3. **Версионирование**: Для сброса кэша у пользователей при внесении правок в CSS/JS используйте теги версии в шаблонах или встроенные механизмы сброса кэша Django.
