# Статические файлы (static)

Каталог для хранения CSS, JavaScript и изображений, используемых во фронтенде.

## Структура

### `css/`
- `style.css` — основные стили сайта (цветовая схема, шрифты, макет).
- `pages.css` — стили для конкретных типов страниц.
- `navigation.css` — стили для меню и навигационных панелей.

### `js/`
- `main.js` — основные скрипты сайта (анимации, обработчики форм).

### `images/`
- Папка для стандартных иконок, фонов и заглушек, которые не загружаются через админку.

## Использование
Файлы из этого каталога подключаются в шаблонах с помощью тега `{% static 'path/to/file' %}`.
