# Шаблоны сайта (Templates)

Этот каталог содержит HTML-файлы, определяющие внешний вид сайта.

## Структура

### 1. Базовые шаблоны (Корень)
- `base.html`: Главный каркас сайта. Содержит общую структуру HTML, подключение стилей, скриптов и блоков `{% block content %}`.
- `header.html`: Шапка сайта (логотип, основное меню).
- `footer.html`: Футер (контакты, ссылки, копирайт).

### 2. Папки приложений
Каждое приложение имеет свою подпапку в `templates/` для хранения специфичных страниц:
- `accounts/`: Вход, регистрация, профиль.
- `main/`: Главная страница, статические страницы.
- `news/`: Списки новостей и детальные страницы статей.
- `portfolio/`: Галерея работ и прайс-листы.
- `reviews/`: Список отзывов и форма добавления.
- `tickets/`: Система тикетов (список, создание, сообщения).

### 3. Компоненты (`includes/`)
Часто повторяющиеся элементы интерфейса:
- Кнопки, карточки, пагинация, блоки уведомлений.

---

## Используемые технологии
- **Django Template Language (DTL)**: Логика отображения (циклы, условия).
- **Bootstrap 5**: Основа верстки и сеточная система.
- **HTMX**: Для динамического обновления частей страницы без перезагрузки (например, в избранном или тикетах).
