# Глобальные шаблоны (templates)

В этом каталоге хранятся HTML-шаблоны сайта.

## Структура

- `base.html` — базовый шаблон, содержащий общую структуру (head, подключение стилей, скриптов). Все остальные шаблоны наследуются от него.
- `header.html` — шапка сайта (логотип, навигация, домен, контакты).
- `footer.html` — подвал сайта (ссылки, копирайт, контактная информация).
- `404.html` — страница для ошибки "Страница не найдена".
- `500.html` — страница для критических ошибок сервера.
- `category-bl.html` — общий блок для отображения категорий.
- `logo_p.html` — блок отображения логотипа.
- `robots.txt` — шаблон для генерации файла robots.txt.

## Подкаталоги

- `accounts/` — шаблоны для логина, регистрации и профиля.
- `admin/` — кастомные шаблоны для переопределения стандартной админки Django.
- `tickets/` — шаблоны для системы техподдержки.
- `includes/` — повторяющиеся фрагменты кода (например, пагинация).
